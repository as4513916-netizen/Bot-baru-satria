# Bot-baru-satria
🤖 Bot WhatsApp Pengubah Link TikTok & YouTube

## ✨ Fitur Utama
✅ Kirim link TikTok → Otomatis jadi Catatan Suara (VN)
✅ Kirim link YouTube + tulis `mp3` → Jadi VN/MP3
✅ Kirim link YouTube saja → Jadi Video MP4
✅ Ketik `menu` untuk lihat panduan

## 🛠️ Cara Memasang
1. Pasang Node.js, FFmpeg, dan yt-dlp
2. Unduh kode lalu buka folder di terminal
3. Jalankan perintah:
```bash
npm install
npm start
